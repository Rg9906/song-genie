# 🤖 Ollama Setup for Dynamic Question Framing

## 🎯 What This Gives You

Instead of template-based framing, you can get **truly dynamic questions** generated by AI models like:
- Llama 3
- Mistral  
- Phi-3
- And many more free models

## 🚀 Quick Setup

### **Step 1: Install Ollama**
```bash
# Windows (PowerShell)
iwr -useb https://ollama.com/install.ps1 | iex

# Or download from https://ollama.com
```

### **Step 2: Start Ollama**
```bash
# It runs as a service, no need to start manually
# Just make sure it's running: http://localhost:11434
```

### **Step 3: Download a Free Model**
```bash
# Small, fast model (good for questions)
ollama pull phi3

# Medium model (better quality)
ollama pull llama3:8b

# Large model (best quality)
ollama pull llama3:70b
```

### **Step 4: Enable LLM in Your Code**
Edit `backend/logic/simple_enhanced.py`:
```python
# Change this line:
self.llm_framer = FreeLLMQuestionFramer(use_llm=False)

# To this:
self.llm_framer = FreeLLMQuestionFramer(use_llm=True)
```

### **Step 5: Test It**
```bash
python test_diverse_questions.py
```

## 🔧 Advanced Setup

### **Custom Prompts**
You can edit `free_llm_question_framer.py` to customize the LLM prompts:

```python
prompt = f"""
You are a friendly music guessing game host.
Rephrase this question in a natural, fun way:

Original: "{base_question['text']}"

Make it sound like you're talking to a friend who loves music!
Keep it short and conversational.
"""
```

### **Model Switching**
```python
# In free_llm_question_framer.py, change the model:
"model": "phi3"           # Fast
"model": "llama3:8b"      # Balanced  
"model": "llama3:70b"     # Best quality
```

## 🚨 Troubleshooting

### **Ollama Not Running**
```bash
# Check if running:
curl http://localhost:11434/api/tags

# Should return model list
```

### **Model Not Downloaded**
```bash
# List available models:
ollama list

# Download if missing:
ollama pull phi3
```

### **Performance Issues**
- Use **phi3** for faster responses
- Use **llama3:8b** for better quality
- Close other apps if using large models
